EMC: Reebok Storage

A distribution facility with climate-controlled storage, loading docks and the warehouse-management systems to run it.

Scope

We built a distribution facility for Elite Merchandising Company (EMC) in the Industrial Area of 6th of October City. It was a full EPC turnkey contract, finished in 2023. The building runs on pre-engineered steel, with climate-controlled storage areas, loading docks built for the job, and material-handling systems threaded through it. That's what an apparel distribution operation needs to move stock.

We ran it from design through construction, so one team was responsible for the whole thing. We put in the warehouse-management infrastructure, the security systems, and a logistics layout planned around how goods come in, sit in storage and go back out. The building gives EMC the throughput and environmental control a global brand asks for.

Challenge

Build a climate-controlled distribution hub with loading docks and material-handling systems to a global brand's standards, all on one turnkey contract.

Solution

Full EPC delivery: pre-engineered steel, warehouse-management infrastructure, security systems and a logistics layout built for flow, all from design through construction.

Result

A fully equipped storage and distribution facility, handed over in 2023 and running EMC's day-to-day distribution.
